# 🏦 Projeto 02 — Sistema de Banco

## Descrição
Sistema bancário com diferentes tipos de conta, transações e extrato.

## Conceitos aplicados
- Herança e Polimorfismo
- Classes Abstratas
- Exceções customizadas
- Encapsulamento

## Funcionalidades
- ✅ Conta Corrente (com cheque especial)
- ✅ Conta Poupança (com rendimento)
- ✅ Conta Investimento (com rentabilidade)
- ✅ Depósito, saque e transferência
- ✅ Extrato completo com histórico
- ✅ Aplicar rendimento mensal

## Como rodar
```bash
cd projetos/02-sistema-de-banco
dotnet run
```
