# ⚔️ Projeto 03 — RPG: Batalha por Turnos

## Descrição
Jogo de RPG com batalha por turnos, classes de personagens e habilidades especiais.

## Conceitos aplicados
- Classes abstratas e Herança
- Interfaces para habilidades
- Enums para estados e tipos
- Polimorfismo extensivo
- Random e simulação

## Personagens
- **Guerreiro** — alto HP, dano físico, habilidade: Golpe Brutal
- **Mago** — baixo HP, dano mágico alto, habilidade: Bola de Fogo
- **Arqueiro** — velocidade alta, ataques duplos, habilidade: Chuva de Flechas
- **Curandeiro** — pode curar aliados, habilidade: Cura Divina

## Como rodar
```bash
cd projetos/03-rpg-batalha-por-turnos
dotnet run
```
