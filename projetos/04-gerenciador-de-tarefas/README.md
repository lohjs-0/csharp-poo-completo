# ✅ Projeto 04 — Gerenciador de Tarefas

## Descrição
Sistema de gerenciamento de tarefas com prioridade, status, categorias e relatórios.

## Conceitos aplicados
- Generics e Coleções
- Delegates e Eventos
- Interfaces (IComparable, IEnumerable)
- Enums
- LINQ

## Funcionalidades
- ✅ Criar, editar e remover tarefas
- ✅ Prioridade (Alta, Média, Baixa)
- ✅ Status (Pendente, Em Progresso, Concluída, Cancelada)
- ✅ Categorias personalizáveis
- ✅ Prazo e notificação de atraso
- ✅ Filtros e ordenação por qualquer campo
- ✅ Estatísticas e relatório de produtividade

## Como rodar
```bash
cd projetos/04-gerenciador-de-tarefas
dotnet run
```
