# Exercício 05 — Construtor com Validação

## 🎯 Objetivo
Praticar construtores com regras de validação robustas.

## 📋 Requisitos
Crie a classe `Funcionario`:
- **Propriedades**: Nome, CPF, Salario, DataAdmissao, Cargo
- **Validações no construtor**: Nome não vazio, CPF com 11 dígitos, Salário > 0
- **Métodos**: `CalcularSalarioAnual()`, `TempoDeEmpresa()` (em meses), `ExibirFicha()`, `AplicarAumento(double percentual)`
- **Bônus**: Valide o CPF com dígitos verificadores
