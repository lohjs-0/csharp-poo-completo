# Exercício 10 — Lista de Contatos

## 📋 Requisitos
Crie classes `Contato` (Nome, Telefone, Email, Grupo) e `AgendaContatos`. A agenda deve suportar: `AdicionarContato`, `RemoverContato`, `BuscarPorNome`, `BuscarPorGrupo`, `ListarOrdenados` (alfabético), `ExibirAgenda`. Bônus: implemente exportação para formato de texto.
