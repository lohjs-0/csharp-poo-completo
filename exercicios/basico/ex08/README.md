# Exercício 08 — Classe Livro

## 🎯 Objetivo
Praticar modelagem de entidades e formatação de saída.

## 📋 Requisitos
Classe `Livro` com Titulo, Autor, ISBN, AnoPuplicacao, NumeroPaginas, Genero. Métodos: `EhAntigo()` (mais de 50 anos), `IdadeDoLivro()`, `ExibirFicha()`. Crie uma classe `Biblioteca` que gerencie uma coleção de livros com métodos `AdicionarLivro()`, `BuscarPorAutor()`, `BuscarPorGenero()`, `ExibirCatalogo()`, `LivroMaisAntigo()`.
