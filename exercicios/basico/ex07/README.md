# Exercício 07 — Classe Produto com Desconto

## 🎯 Objetivo
Praticar propriedades calculadas e lógica de negócio.

## 📋 Requisitos
Classe `Produto` com Nome, Preco, Categoria, Estoque. Métodos: `AplicarDesconto(double percentual)`, `PrecoComDesconto(double percentual)`, `AdicionarEstoque(int qtd)`, `RemoverEstoque(int qtd)`, `EstaDisponivel()`, `ExibirInfo()`.
