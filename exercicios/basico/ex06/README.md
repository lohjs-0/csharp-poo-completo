# Exercício 06 — Classe Aluno com Média

## 🎯 Objetivo
Trabalhar com listas internas e cálculos.

## 📋 Requisitos
Classe `Aluno` com propriedades Nome e Matrícula (auto-gerada), lista de notas, métodos `AdicionarNota(double nota)`, `CalcularMedia()`, `ObterMaiorNota()`, `ObterMenorNota()`, `ExibirBoletim()`. A matrícula deve ser gerada automaticamente e incrementada.
