# Exercício Intermediário 10

## Sistema de Desconto com Strategy Pattern
Interface `IEstrategiaDesconto` com `CalcularDesconto(Pedido p)`. Implemente: `DescontoPercentual`, `DescontoFixo`, `DescontoProgressivo` (mais itens = mais desconto), `DescontoClienteVIP`, `DescontoCombinado`. Um `Carrinho` que aceita e aplica qualquer estratégia.
