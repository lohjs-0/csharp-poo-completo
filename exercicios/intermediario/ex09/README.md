# Exercício Intermediário 09

## Tratamento de Exceções Customizadas
Crie uma hierarquia de exceções: `SistemaException → ValidacaoException`, `NegocioException`, `SegurancaException`. Implemente um sistema de login com: `UsuarioNaoEncontradoException`, `SenhaIncorretaException`, `ContaBloqueadaException`, `SessaoExpiradaException`. Demonstre tratamento específico de cada uma.
