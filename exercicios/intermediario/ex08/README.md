# Exercício Intermediário 08

## Coleção Genérica de Alunos
Crie `Turma<T>` onde T é `Aluno`. A turma deve: adicionar/remover alunos, calcular média da turma, retornar aprovados/reprovados, listar ordenados por nota. Use `IEnumerable<T>`, `IComparable<T>`. Implemente também um `RelatorioTurma` que gera estatísticas completas.
