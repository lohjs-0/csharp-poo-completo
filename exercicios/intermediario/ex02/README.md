# Exercício Intermediário 02

## Polimorfismo com Formas Geométricas
Crie a classe abstrata `Forma` com métodos abstratos `CalcularArea()` e `CalcularPerimetro()`. Implemente: `Circulo`, `Retangulo`, `Triangulo`, `Trapezio`. Crie um método estático que receba `List<Forma>` e retorne estatísticas (total de formas, área total, maior área, menor perímetro).
