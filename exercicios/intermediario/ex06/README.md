# Exercício Intermediário 06

## Interface: IComparavel para Produtos
Implemente `IComparable<Produto>` na classe `Produto` para ordenação por preço. Implemente também `IEquatable<Produto>` para comparação por ISBN/código. Crie um `Catalogo` que permita ordenar por diferentes critérios (preço, nome, avaliação) usando `IComparer<T>`.
