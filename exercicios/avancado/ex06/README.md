# Exercício Avançado 06 — Decorator Pattern

## 🎯 Objetivo
Adicionar comportamentos dinamicamente usando o padrão Decorator.

## 📋 Requisitos
- Interface `ITextoProcessor` com `Processar(string texto)`
- Implementação base: `ProcessadorBase`
- Decorators: `DecoratorMaiusculas`, `DecoratorRemoveEspacos`, `DecoratorAddTimestamp`, `DecoratorCriptografa`, `DecoratorContaPalavras`
- Permita combinar qualquer sequência de decorators

## 💡 Conceitos
Decorator Pattern, Composição, Interfaces
